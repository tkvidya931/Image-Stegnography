# Image Steganography using LSB Technique

This project implements Image Steganography using the Least Significant Bit (LSB) technique in the C programming language. The application securely hides secret data inside a BMP image by modifying the least significant bits of image pixel values using bitwise operations.

The project supports both encoding and decoding processes:

Encoding: Embeds secret text or files into a BMP image without visibly affecting image quality.
Decoding: Extracts the hidden data from the stego-image and reconstructs the original secret file after validation.
